## Cloud Empress Module for Foundry VTT (Unofficial)

An unofficial implementation of Cloud Empress, an ecological science-fantasy RPG powered by the Panic Engine, for FoundryVTT. Cloud Empress is created by watt and published by worlds by watt.

This Foundry VTT system is adapted from the [MoSh Unofficial Mothership module](https://github.com/Futil/foundry-mothership).

**Official Website**: https://cloudempress.com

#### Features
- Full Cloud Empress core rulebook support
- Character sheets for all 4 jobs: Sellsword, Lordling, Magician, and Courier
- Basic character sheet automation for checks, saves, panic, wounds, and stress
- Basic character generation with class and age selection
- Compendiums for crests, spells, skills, curses, spores, and items
- Rolltables from the core rulebook including names, aliases, equipment, job gear, and more
